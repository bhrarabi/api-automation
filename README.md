# 🚀 Loan Allocation API Automation

Automated API testing framework for validating **loan allocation services** using **Java, Spring Boot, Rest Assured, and TestNG**.

This project is designed to ensure the **correctness, reliability, and stability of loan allocation APIs** by automating functional and regression tests for critical financial workflows.

---

# 📌 Overview

Loan allocation services are responsible for distributing funds to eligible users based on predefined business rules.  
Testing these services manually can be slow and error-prone, especially when multiple scenarios and edge cases exist.

This project provides an **automated API testing framework** that:

- Validates loan allocation logic
- Ensures API responses match expected business rules
- Automates regression testing
- Improves release confidence for financial services

---

# 🧰 Tech Stack

- **Java**
- **Spring Boot**
- **Rest Assured**
- **TestNG**
- **Maven**

---

# ⚙️ Features

✔ Automated API testing for loan allocation services  
✔ Validation of response status, schema, and business logic  
✔ Structured and reusable test architecture  
✔ Support for multiple test scenarios and edge cases  
✔ Easy integration into CI/CD pipelines  
✔ Clear test reports for debugging and analysis
